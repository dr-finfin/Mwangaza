
Mwangaza
A video-centric learning platform that will be used to implement the Kenyan Competency-Based Education (CBE) curriculum with the help of Artificial Intelligence for project guidance.

Tech Stack
Client-Side: 
    React 19
    Vite
    Tailwind CSS

Server-Side: 
    Node.js
    Express

AI: 
    Google Gemini API

Database: 
    better-sqlite3
    MongoDB

Installation
To run the application, the following steps need to be followed:

    Clone the project from the provided repository URL.
    Run the following command to install all the dependencies:
    npm install
    Next, create a new file with the name ".env" and add the following variables:
    GEMINI_API_KEY= Your Google Gemini API key
    APP_URL= URL where the application will be hosted
    MONGO_URI= Your database connection URL
    JWT_SECRET= Any random string that will be used as a secret key
